Usage Instructions
This application supports word patterns to find words in specific format from the English Dictionary.
  • Word Length: Enter word length and get all words of same length from the dictionary.
  • Word Search Pattern: Use '-'(Hyphen, minus or dash) to specify unknown letters.For example A-AC-S. Letters and Hyphen only are allowed.
